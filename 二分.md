## 模板

### 区别点

* `r = mid` or `l = mid` :question:
*  `l + r >> 1` or `l + r + 1 >> 1` :question:

### 1. 满足xxx的第一个位置 / 下界

```c++
class Solution {
public:
    int search(vector<int>& nums) {
        int l = 0, r = nums.size() - 1;
        while (l < r) {
            int mid = l + r >> 1;
            if (满足xxx) r = mid;
            else l = mid + 1;
        }
        return l;
    }
};
```



### 2. 满足xxx的最后一个位置 / 上界

```c++
class Solution {
public:
    int search(vector<int>& nums) {
        int l = 0, r = nums.size() - 1;
        while (l < r) {
            int mid = l + r + 1 >> 1;
            if (满足xxx) l = mid;
            else r = mid - 1;
        }
        return l;
    }
};
```
